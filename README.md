## 🆕 مميزات النسخة 2.0
- 🏆 **وضع البطولة** - 4 فرق، نصف نهائي + نهائي
- 🥅 **ركلات الترجيح** - وضع مستقل مع حارس ذكي
- 🧤 **حارس مرمى ذكي** - يتحرك ويصد تلقائياً
- 🟨 **نظام البطاقات** - أصفر وأحمر مع الأخطاء
- ⚙️ **4 تشكيلات** - 4-3-3 / 4-4-2 / 5-3-2 / 3-5-2
- 🗺️ **خريطة Mini-Map** في زاوية الشاشة
- 🎽 **8 فرق** قابلة للاختيار مع تقييمات
- 📊 **إحصائيات تفصيلية** - تسديدات، استحواذ، تصديات
