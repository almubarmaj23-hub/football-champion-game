## 🎮 مميزات اللعبة

- ⚽ **ملعب ثلاثي الأبعاد** مرسوم بالكامل بـ Canvas
- 🤖 **ذكاء اصطناعي متطور** بـ 3 مستويات صعوبة
- 👥 **11 لاعب في كل فريق** مع حركة ديناميكية
- 🎯 **فيزياء الكرة الواقعية** مع الاحتكاك والتسارع
- 🌟 **تأثيرات بصرية مذهلة** - انفجارات عند التسجيل
- 🎵 **مؤثرات صوتية** باستخدام Web Audio API
- 📱 **متوافق مع الجوال** - دعم اللمس والأزرار
- 🏆 **نظام نقاط** مع جدول المتصدرين
- ⏱️ **وقت رسمي** 90 دقيقة مع الشوطين
- 🎨 **واجهة احترافية** تشبه FIFA وeFootball

## 🕹️ أزرار التحكم

| الزر | الوظيفة |
|------|---------|
| `←` `→` `↑` `↓` | تحريك اللاعب |
| `Space` أو `Z` | تمرير / ضربة |
| `X` | تدخل / سرقة الكرة |
| `C` | تمرير طويل / عرضية |
| `Shift` | الجري السريع |
| `Enter` | إيقاف مؤقت |

## 🛠️ التقنيات المستخدمة

- **HTML5 Canvas API** - رسم الملعب واللاعبين
- **JavaScript ES6+** - منطق اللعبة والذكاء الاصطناعي
- **Web Audio API** - المؤثرات الصوتية
- **CSS3 Animations** - واجهة المستخدم والرسوم المتحركة
- **Responsive Design** - التوافق مع جميع الأجهزة
